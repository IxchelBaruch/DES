Ultima actualizacion: 25/03/2022

---OBJETIVO---
Crear un videojuego para consola Xbox, estilo metroidvania, en formato 2D.

---METODOLOGÍA---
La metodologia utilizada para el equipo de desarrollo fue de tipo SCRUM, preguntando y revisando semana a semana, a cada uno de los integrantes, por sus tareas asignadas.


---HERRAMIENTAS UTILIZADAS---
Unity, como principal motor de desarrollo.
Fmod, como motor para el diseño y creación de audios.

---ARQUITECTURA DEL PROYECTO---


---TRABAJO EN CONJUNTO CON LA CONSOLA---
El desarrollo del videojuego se realizo en Unity, para generar la version para consola fue necesario instalar la extension "Universal Windows Platform" la cual nos permite exportar el proyecto para una version para consola.

Tambien, fue necesario adquirir un development tool para la consola de Xbox, para poder probar el juego.


---MOTIVACIÓN---
 El motivo del desarrollo de este proyecto, surge como necesidad de crear un proyecto semestral en la universidad, sin embargo, con el pasar del desarrollo nos motivamos a entregar y crear un videojuego de calidad y con un buen diseño y programacion.

El diseño del videojuego surge a partir de una lluvia de ideas con la participacion de todos los integrantes del equipo de desarrollo. Dando como resultado una ambientacion estilo japonesa con diseños de personajes mitologicos de diferentes partes del mundo.

---CONTRIBUCIÓN--- 
 Para desarollar este proyecto tuvimos que aprender a utilizar el "nuevo input system" de Unity, asi como, Fmod, un programa para creacion de audio.
